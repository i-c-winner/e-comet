# Тестовое задание

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

Эта команда также автоматически выполнит `npm run prepare`, который настроит Husky для git hooks.

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173` (по умолчанию).

### 3. Сборка

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`.

### 4. Предварительный просмотр сборки

```bash
npm run preview
```

Запустит локальный сервер для просмотра production сборки.
